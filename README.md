# ToneGuard

AI-powered Chrome extension that detects the tone of your messages in real-time and suggests better rewrites — before you hit send.

Works on Gmail, Slack, LinkedIn, Twitter, WhatsApp Web and any website with a text box.

---

## The Problem

We all send messages we regret. An angry email to your manager. A passive-aggressive reply to a colleague. A sarcastic comment that ruins a relationship.

ToneGuard catches these moments before they happen.

---

## How It Works

1. You type a message anywhere on the web
2. ToneGuard detects the tone automatically after 2.5 seconds
3. If the tone is negative it shows 3 professional rewrites
4. Click "Use this" to replace your message instantly

---

## Tones Detected

- Anger
- Passive Aggressive
- Sarcasm
- Warmth
- Neutral
- Assertive

---

## Install ToneGuard

**Step 1 — Download the code**

Click the green **Code** button on this page → Download ZIP → Extract it

**Step 2 — Open Chrome extensions**

Go to `chrome://extensions` in Chrome

**Step 3 — Enable Developer Mode**

Toggle **Developer mode** ON in the top right corner

**Step 4 — Load the extension**

Click **Load unpacked** → Select the `extension` folder from the extracted ZIP

**Step 5 — Pin it**

Click the puzzle piece icon in Chrome toolbar → Pin ToneGuard

**Step 6 — Done**

Open Gmail and type any message. Wait 2.5 seconds. ToneGuard appears automatically.
